Programme réalisé par Cindie Scheuren et Jacques Hogge
Octobre 2020

la premiere fonction permet de voire si la liste de communes est ecritre dans l'ordre alphabétique, si c'est pas le cas,
le programme retourne False

la deuxieme fonction permet de trouver les coordonnées d'un commune parmis la liste de toutes les communes

la troisème fonction permet de calculer la distance entre 2 communes

la quatrième fonction permet de calculer la distance total a parcourire pour faire un tour de toute les communes définie pas une liste

nous n'avons pas rencontré de probleme durant la réalisation de ce programme
